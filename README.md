# Axé Hero

Projeto acadêmico inspirado em Guitar Hero, desenvolvido com Arduino e adaptado para músicas de axé.

## Sobre o projeto

O Axé Hero é um jogo físico baseado em ritmo, no qual notas descem por trilhas de LEDs NeoPixel e o jogador deve pressionar o botão correto no momento certo.

O projeto foi idealizado por mim e por Matheus Sobral, utilizando hardware físico, programação embarcada e lógica de jogo em tempo real.

## Músicas utilizadas

- Faraó
- Baianidade Nagô
- Selva Branca

## Tecnologias utilizadas

- Arduino UNO
- C/C++ com Arduino
- Fitas LED NeoPixel
- LCD I2C 16x2
- Buzzer
- Botões físicos
- Interrupções
- millis()
- Biblioteca Adafruit NeoPixel
- Biblioteca LiquidCrystal_I2C

## Funcionalidades

- Seleção de música pelo LCD
- Seleção de dificuldade
- Notas descendo pelas trilhas de LED
- Pontuação
- Sistema de vidas
- Combo
- Sons de acerto, erro, início e fim de jogo
- Interface visual no LCD
